# FixSmart  
**Smart IT Helpdesk & Office Support System**

FixSmart adalah aplikasi **IT Helpdesk berbasis web** yang digunakan untuk mencatat, mengelola, dan memantau **laporan gangguan IT** di lingkungan kantor atau instansi.  
Aplikasi ini dirancang sederhana, ringan, dan mudah digunakan untuk kebutuhan internal.

---

## 🚀 Fitur Utama
- 📋 Laporan gangguan IT (hardware, software, jaringan, dll)
- 🛠️ Manajemen tiket helpdesk
- 🔔 Notifikasi otomatis ke **Telegram**
- 📊 Monitoring dan dokumentasi permasalahan IT
- 🌐 Berbasis web (PHP)

---

## 🎯 Tujuan Aplikasi
FixSmart dibuat untuk membantu:
- Tim IT dalam mengelola laporan gangguan
- Pencatatan masalah IT secara terstruktur
- Respon gangguan IT yang lebih cepat dan terdokumentasi
